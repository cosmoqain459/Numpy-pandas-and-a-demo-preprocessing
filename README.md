# Data Science Projects

A collection of data science projects and practice notebooks 
built while learning Python for machine learning.

## Notebooks

### 1. Bihar Flood Data Cleaning Pipeline
End-to-end data cleaning and feature engineering for Bihar flood data,
prepared for 4 machine learning prediction tasks:
- Disaster Severity Classification (Low / Medium / High)
- Affected Population Estimation
- Relief Resource Requirements
- District Risk Priority Ranking

**Dataset:** India Flood Inventory v3 — [Zenodo](https://zenodo.org/records/11275211)

---

### 2. NumPy Practice

### 3. Pandas Practice

groupby, and basic EDA.
